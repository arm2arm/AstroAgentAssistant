# AstroAgent Concept

## When to Use
Use this skill when drafting concept papers, architecture notes, or strategy around AstroAgent.

## Procedure
1. Frame AstroAgent as augmentation-first, not replacement-first.
2. Emphasize provenance, FAIR alignment, and reproducible execution.
3. Connect literature, archives, local datasets, code generation, and workflows.

## Pitfalls
- Avoid describing AstroAgent as a generic chatbot.
- Avoid dropping the human-in-the-loop and provenance-centered framing.

## Verification
- Final concept text includes architecture, data access, reproducibility, and role boundaries.
